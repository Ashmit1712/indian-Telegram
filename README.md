# Indian Telegram

A modern messaging application inspired by Telegram, built for Indian users with localized features and support.

## Features

- 📱 Real-time messaging
- 👥 Group chats and channels
- 🔐 End-to-end encryption
- 📁 File sharing
- 🔔 Push notifications
- 🌍 Multi-language support (Hindi, English, etc.)
- 📞 Voice and video calling
- ✨ Stickers and emojis

## Tech Stack

- **Frontend:** React.js, Redux
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Real-time:** Socket.io
- **Authentication:** JWT

## Installation

### Prerequisites
- Node.js (v14+)
- npm or yarn
- MongoDB

### Setup

1. Clone the repository
```bash
git clone https://github.com/Ashmit1712/indian-Telegram.git
cd indian-Telegram
```

2. Install dependencies
```bash
npm install
```

3. Configure environment variables
```bash
cp .env.example .env
```

4. Start the application
```bash
npm start
```
